# 🚗 RideFleet - Vehicle Rental App

A modern, premium vehicle rental web application built with vanilla **HTML**, **CSS**, and **JavaScript**. Browse, filter, and book vehicles at affordable hourly rates.

> **Live Demo:** [https://madankumar-hub.github.io/Car-Rental-Website/](https://madankumar-hub.github.io/Car-Rental-Website/)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🚘 **12+ Vehicles** | Cars, SUVs, Bikes, Luxury & Electric vehicles |
| 🔍 **Smart Filters** | Filter by All, Cars, SUVs, Bikes, Luxury, Electric |
| 💰 **Hourly Pricing** | Transparent pricing from ₹80/hr to ₹3,500/hr |
| 📋 **Instant Booking** | Book with name, phone, date, time & duration |
| 💵 **Auto Price Calc** | Total price auto-calculated based on hours selected |
| 📱 **Fully Responsive** | Works on Desktop, Tablet & Mobile |
| 🎨 **Modern Dark UI** | Premium dark theme with orange gradient accents |
| ✨ **Smooth Animations** | Scroll reveals, hover effects & micro-animations |

## 🛠️ Tech Stack

- **HTML5** — Semantic structure & SEO-friendly markup
- **CSS3** — Custom properties, Flexbox, Grid, animations, glassmorphism
- **JavaScript (ES6+)** — DOM manipulation, dynamic rendering, event handling

> No frameworks or libraries used — 100% vanilla code!

## 🚀 How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/madankumar-hub/Car-Rental-Website.git
   ```

2. **Open in browser**
   ```bash
   cd vehicle-rental
   open index.html
   ```
   Or simply double-click `index.html` to open it in your default browser.

## 📸 Sections

### 🏠 Hero Section
- Eye-catching landing with animated background shapes
- Search bar for pickup location
- Stats: 500+ Vehicles, 10K+ Riders, 50+ Cities, 4.8★ Rating

### 🚘 Vehicle Grid
- Filter buttons to browse by category
- Each card shows: name, brand, seats, fuel type, transmission, mileage, rating, trips & price/hr
- "Book Now" and "Details" buttons on every card

### 📋 Booking Modal
- Full booking form: Name, Phone, Date, Time, Duration
- Auto-calculated total price
- Success toast notification on booking

- Call Now & Email Us buttons
- Location, Phone, Email & Working Hours info
